This project analyzes 3,900 customer purchase records to understand shopping behavior and spending patterns.
The dataset includes demographics, product details, discounts, subscriptions, and review ratings.
Data cleaning and preprocessing were performed using Python (Pandas).
Missing values were handled using median imputation based on product category.
Feature engineering was applied to create age groups and customer segments.
The cleaned data was stored and analyzed using PostgreSQL.
Business insights such as revenue by gender, subscription impact, and top products were derived.
An interactive dashboard was built using Microsoft Power BI for visualization.
The project provides actionable recommendations for marketing, loyalty programs, and revenue growth.
This end-to-end analysis demonstrates strong data cleaning, SQL, and visualization skills.
